# 企业融资服务网站 - 规格说明书

## 1. 项目概述

- **项目名称**: 卓越企业融资服务平台
- **项目类型**: 企业服务官方网站（品牌展示 + 获客）
- **核心功能**: 展示公司实力、提供在线融资评估工具、预约咨询
- **目标用户**: 中小企业主（年营收100万以上，一般纳税人，有开票/纳税）

## 2. UI/UX 规格

### 2.1 布局结构

**页面 Sections**:
1. **顶部导航** - 固定导航栏，含Logo、菜单、联系电话
2. **Hero区域** - 全屏首屏，核心价值主张
3. **核心优势** - 4大优势展示
4. **合作银行** - 合作银行Logo展示
5. **服务流程** - 融资服务流程图
6. **在线评估工具** - 贷款额度测算工具
7. **成功案例** - 客户案例展示
8. **预约咨询** - 联系表单
9. **底部信息** - 版权、联系方式等

**响应式断点**:
- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

### 2.2 视觉设计

**色彩系统**:
- 主色(深蓝): #0A1628
- 次主色(蓝): #1E3A5F
- 强调色(金): #C9A962
- 辅助色(浅金): #E8D5A3

- 浅灰背景: #F5F7FA

**字体系统**:
- 中文: "Noto Serif SC", "Source Han Serif CN", serif
- 英文: "Playfair Display", serif

**间距系统**:
- 基础单位: 8px

- 组件间距: 32px

**视觉效果**:
- 卡片阴影: 0 4px 24px rgba(10, 22, 40, 0.08)
- 悬停阴影: 0 8px 32px rgba(10, 22, 40, 0.12)
- 边框圆角: 4px (按钮), 8px (卡片)
- 金色渐变: linear-gradient(135deg, #C9A962, #E8D5A3)

### 2.3 组件

**按钮**:
- 主按钮: 金色背景，深蓝文字
- 次按钮: 透明边框，金色边框

**卡片**:
- 白色背景，轻阴影
- 悬停: 提升效果+阴影加深

**输入框**:
- 浅灰背景，金色聚焦边框
- 标签动画上移

**进度条**:
- 金色渐变进度条

## 3. 功能规格

### 3.1 在线融资评估工具

**输入字段**:
- 企业名称（文本）
- 联系人（文本）
- 联系电话（数字）
- 年营业额（选择：100-300万/300-500万/500-1000万/1000万以上）
- 纳税等级（选择：A级/B级/C级/未评级）
- 是否有开票（单选：是/否）
- 所在行业（选择：化工/食品/农业/建材/其他）

**计算逻辑**:
- 基础额度 = 年营业额 × 系数
- 纳税A: 系数1.5, B: 系数1.2, C: 系数0.8
- 有开票: +20%额度

**输出**:
- 预估贷款额度范围
- 推荐银行方案（模拟）

### 3.2 预约咨询表单

**必填字段**:
- 姓名
- 公司名称
- 联系电话
- 融资需求描述

**功能**:
- 表单验证
- 提交成功提示
- 模拟提交（显示成功信息）
