# 云上融业务定位与边界声明

> **版本:** v1.0 · **生效日期:** 2026-04-25
> 本文件定义本项目的业务定位、产品形态与自我边界,适用于代码、文档、对外
> 文案、客户合同、用户协议。所有功能开发与产品决策需与本声明保持一致。

---

## 一、我们是什么

| 维度 | 定位 |
|------|------|
| **服务对象** | 融资顾问公司、企业服务咨询机构 |
| **产品形态** | 数字化作业系统(B2B SaaS 工具) |
| **业务范围** | 现金流规划咨询的工作台与协同工具 |
| **收入模式** | 软件订阅费(顾问公司付费) |
| **数据角色** | 数据处理者(processor),非数据控制者 |

**一句话**:云上融是给融资顾问公司用的现金流规划作业系统。

---

## 二、我们不做什么(自我禁飞区)

为保持工具属性、避免越界进入需金融牌照的业务范畴,本系统**不**:

1. **不开展征信评级业务** —— 不出具任何形式的客户信用评分作为对外结论
2. **不直接撮合贷款** —— 不与银行/资金方建立放贷撮合的商业关系
3. **不向最终客户承诺额度** —— 系统中所有"参考区间"由顾问录入,非系统判定
4. **不向银行/三方返佣换数据** —— 不以客户数据交换商业利益
5. **不替顾问做信用判断** —— 系统提供资料整理与展示,判断由顾问做出
6. **不冒充金融机构** —— 任何对外文案不暗示本平台为放贷主体或金融服务方

## 三、四条法律底线(无条件遵守)

无论行业标准是否明确,以下四条均为不可触碰的法律红线:

| # | 红线 | 法律依据 | 系统层应对 |
|---|------|---------|-----------|
| 1 | **未授权处理个人信息** | 个人信息保护法 | 顾问上传客户征信/流水须附授权书附件,无授权不可入库 |
| 2 | **数据泄露不通报** | 数据安全法 + 个保法 | 加密存储 + 审计日志 + 数据访问留痕 + 应急流程文档 |
| 3 | **电销骚扰/虚假宣传** | 反诈法 + 广告法 | 电销话术模板审查 + 可挂断 + 不承诺额度 |
| 4 | **冒充金融机构/承诺放贷** | 刑法 + 银保监 | 全产品文案审查,杜绝放贷主体暗示 |

---

## 四、不在本平台边界内的合规义务

以下义务由顾问公司(数据控制者)承担,不在本平台责任范围:

- 顾问与企业客户之间的咨询服务合同与履约
- 顾问代客户对接银行的助贷合规
- 客户最终融资结果的真实性与时效性
- 顾问对客户作出的具体融资建议

---

## 五、命名与话术规范

为保持定位一致,以下表述是本项目内的**强制规范**:

| ❌ 禁用 | ✅ 推荐 |
|--------|--------|
| 信用评分 / credit score | 资料完整度评分 / 顾问诊断评分 |
| 推荐银行 / 智能匹配 | 历史案例参考 / 案例库 |
| 评估额度 / 系统额度 | 顾问参考区间(顾问录入) |
| 智能诊断 | 资料整理工作台 |
| 我们能贷到 XX | 帮你梳理资料、对接合适的银行产品 |
| 撮合 / 放贷 / 担保 | 咨询 / 顾问 / 工具 |
