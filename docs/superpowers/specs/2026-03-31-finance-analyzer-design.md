# 企业融资数据智能分析工具 — 设计文档

## 概述

面向贷款中介/企业融资服务公司的数据分析工具。帮助融资顾问（非技术用户）上传客户征信报告和银行流水，自动解析并输出结构化分析数据，辅助出具融资方案。

## 用户画像

- 融资顾问，非技术人员
- 需要极简界面：拖拽上传 → 自动分析 → 查看/导出结果
- 日常工作流程：收到客户材料 → 分析征信和流水 → 出融资方案

## 技术架构

### 架构选型：前后端分离

- **前端**：React 18 + TypeScript + Ant Design 5
- **后端**：Python FastAPI
- **数据库**：SQLite（本地轻量存储，按客户组织）
- **部署**：本地运行

### 系统架构

```
用户浏览器 (React + Ant Design)
    │
    │  HTTP/REST API
    ▼
FastAPI 后端
    ├── /api/credit-report/upload    征信报告上传与解析
    ├── /api/bank-statement/upload   银行流水上传与解析
    ├── /api/analysis/{client_id}    获取整合分析结果
    └── /api/export/{client_id}      导出Excel/PDF
    │
    ├── 解析引擎
    │   ├── PDF文本提取 (pdfplumber)
    │   ├── 图片OCR (Tesseract + Pillow + pdf2image)
    │   └── Excel解析 (pandas + openpyxl)
    │
    ├── 分析引擎
    │   ├── 征信数据结构化
    │   ├── 流水汇总统计（含去重逻辑）
    │   └── 异常交易检测
    │
    └── 数据存储
        └── SQLite（按客户ID组织）
```

### 关键设计决策

1. **SQLite而非文件存储** — 按客户ID组织数据，方便查询和导出，无需安装额外数据库
2. **解析与分析分离** — 上传后先提取原始数据存储，再按需运行分析，支持重新分析
3. **客户维度管理** — 每个客户一个案例，可关联多份征信报告和多份银行流水

## 功能模块

### 1. 征信报告解析

**输入格式：**
- 人行征信中心导出的PDF（标准格式，规则解析）
- 纸质报告扫描件/手机拍照（JPG/PNG，Tesseract OCR）

**提取数据结构：**

| 类别 | 字段 |
|---|---|
| 负债总览 | 总负债、总余额 |
| 机构明细 | 各类贷款机构数量明细（银行、消费金融、小贷等） |
| 信用卡汇总 | 总额度、已用额度、整体使用率（已用/总额度） |
| 在贷机构 | 当前有贷款的机构列表及各机构贷款数据 |
| 逾期记录 | 逾期次数、金额、时间 |
| 查询记录 | 近1月/近3月/近半年/近1年的贷款审批次数、法人审查次数 |

### 2. 银行流水分析

**输入格式：**
- 银行网银导出的Excel/CSV

**分析输出：**

| 类别 | 具体指标 |
|---|---|
| 收支汇总（原始） | 月均收入、月均支出、月均净利润、总收入、总支出 |
| 收支汇总（去重） | 去重后总收入、去重后总支出、去重后月均收入、去重后月均支出 |
| 资金流向 | 主要收入来源TOP5（对手方+金额占比）、主要支出类别TOP5 |
| 异常交易 | 大额交易、规律性资金调动、频繁整数金额进出 |
| 余额分析 | 月末余额变化趋势、最低余额、平均余额 |
| 交易频次 | 月均交易笔数、日均交易笔数 |

**去重规则：**
1. 同名转账 — 转账对手方姓名与账户持有人相同（自己转自己）
2. 微信/支付宝提现 — 摘要中包含"微信提现"、"支付宝提现"等关键词

去重后输出两组数据供对比：原始数据和去重后的真实经营数据。

**异常交易检测规则：**
- 大额交易：单笔金额 > 月均收入 × 2
- 规律性资金：同一对手方、相同金额、固定周期的转入转出
- 整数交易：大额整数金额（如10万、50万）的频繁进出

### 3. 工作流

向导式三步流程：
1. **上传征信报告** → 自动解析 → 预览提取结果
2. **上传银行流水** → 自动分析 → 预览分析结果
3. **查看分析报告** → 在线浏览 → 导出Excel/PDF

## UI设计

### 整体布局

- 顶部进度条引导三步流程
- 每步一个独立页面，底部"下一步"按钮

### 步骤1：上传征信报告

- 拖拽上传区（支持PDF和图片）
- 上传成功后显示文件信息和解析状态
- 解析完成后预览关键指标卡片（总负债、逾期记录、查询次数等）

### 步骤2：上传银行流水

- 拖拽上传区（支持Excel/CSV）
- 支持上传多份流水（不同银行）
- 解析完成后预览收支汇总

### 步骤3：分析报告

- Tab切换：征信概况 / 银行流水 / 异常交易 / 综合评估
- 右上角导出按钮（Excel / PDF）
- 征信概况：负债明细表、信用卡汇总、在贷机构列表、查询记录
- 银行流水：原始与去重收支对比、月度收支趋势图、资金流向
- 异常交易：异常交易列表（日期、金额、对手方、异常类型）
- 综合评估：核心指标汇总

## 导出结构

### Excel导出（多Sheet）

| Sheet | 内容 |
|---|---|
| 客户概览 | 客户姓名、报告日期、核心指标摘要 |
| 征信详情 | 负债总览、信用卡汇总、在贷机构、逾期记录、查询记录 |
| 流水汇总 | 原始收支 + 去重收支、月度收支明细表 |
| 异常交易 | 异常交易列表（日期、金额、对手方、异常类型） |

### PDF导出

- 封面（客户信息+日期）
- 征信概况（表格+关键指标）
- 流水分析（表格+月度趋势图）
- 异常交易清单
- 综合评估摘要
